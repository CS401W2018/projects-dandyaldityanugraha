CSS Selector

  1. List out the different ways to select an item.
    2. ID Selector: Selects an element based on its unique ID. Example: #myID { color: blue; } selects the element with id="myID".

    Class Selector: Selects elements with a specific class attribute. Example: .myClass { color: green; } selects all elements with class="myClass".

  2. What is the difference between an ID and a Class?
    3. ID: using # can't reuse - noun. Class - . - can reuse - adjective
  3. What is the difference between a SPAN tag and a DIV tag?
    4. SPAN tag: Inline element, does not start a new line. DIV tag: block-level element, start a new line.
  4. What is the difference between a DIV tag and HEADER, MAIN, FOOTER?
    5. DIV tag: Used purely to styling or layout purpose. Header, Main, Footer: Represent introductory, main content, and footer for copy right.
  5. What are different ways to select a color?
    6. Name color, hexadecimal number, or RGB
