  2. Explain the difference between an absolute link and a relative link. Why would you use relative links?

    3. Absolute Link: An absolute link contains the complete URL, including the protocol (http:// or https://), domain name, and the path to the resource.

    Relative Link: A relative link refers to a resource by specifying the path relative to the current document's location.

    Why Use Relative Links? Relative links are beneficial when linking to resources within the same website.
